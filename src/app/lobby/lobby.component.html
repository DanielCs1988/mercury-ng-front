Welcome to Mercury App!

Mercuy is a social media app using Angular 6 and 4 + 1 microservices with multiple backend technologies, including my own Java microframework, Spring, NodeJS, GraphQL and Websockets. Please log in if you want to check out the website!

Alternatively, you can check out the repository for:

  • The Angular frontend, the one you are currently looking at. :-)
  • The backend behind the social feed and users. It's NodeTS with GraphQL-Yoga framework.
  • Or maybe you are interested in my homegrown Java framework? Then the events API is your place!
  • Not a Java person? Then maybe you will find the NodeTS/Socket.io server behind the chat feature to your liking.
  • Want to see an enterprise Java Spring app in action instead of all this? Here you go!
